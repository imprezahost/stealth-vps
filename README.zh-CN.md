## 简介 (占位)

`stealth-vps` 是一个用于在受限网络环境下搭建隐私 VPS 的可复用工具集。

- 基于 Ansible / cloud-init,可重复部署
- 包含 VLESS-Reality + Hysteria2 + 3X-UI 面板
- 默认启用 BBR、fail2ban、UFW 等加固
- 内置 Prometheus + Grafana 可观测性
- MIT 协议(非 AGPL),允许商业使用
