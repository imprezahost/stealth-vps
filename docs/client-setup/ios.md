Recommended clients:

- **Shadowrocket** — paid, App Store, supports VLESS-Reality and Hysteria2
- **Streisand** — paid, App Store, sing-box based
- **Hiddify** — free, App Store, cross-platform

## Importing your config

The `stealth-vps` install writes connection URIs to `/root/stealth-vps-credentials.txt`. Either copy the URI or scan the QR code from the same file.

## Note on App Store availability

In some regions these apps are removed from the local App Store. You may need a different Apple ID region to install them. This is an App Store policy issue, not something the project can work around.
