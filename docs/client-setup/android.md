## Importing your config

The `stealth-vps` install writes connection URIs to `/root/stealth-vps-credentials.txt`. You'll have one `vless://...` URI for Reality and one `hy2://...` URI for Hysteria2.

You can:

1. Copy the URI text and paste into the client (`Import from clipboard`).
2. Scan a QR code (also written next to the URI in the credentials file).
3. Use the 3X-UI panel's subscription link (single URL containing all configured inbounds).
